# Real-Time Trading Dashboard

A real time crypto trading dashboard built to handle high frequency data.

## Frontend Setup

**Prerequisites:** Node.js

1. **Cloning the repo:**
    `git clone https://github.com/amankalra4/Real-Time-Trading-Dashboard.git`

2. **Installing dependencies:**
   `yarn install`

3. **Start the development server:**
   `yarn dev`
   And the app will run at `http://localhost:5173`*

## Backend Setup

**Prerequisites:** bun

1. **Cloning the repo:**
    `git clone https://github.com/saxenanickk/socket-custom-load.git`

2. **Installing dependencies:**
    `bun install`

3. **Start the development server:**
    `bun start`

## Frontend Tech Stack
* **Framework:** React + TypeScript
* **Build Tool:** Vite
* **Styling:** Tailwind CSS
* **State Management:** Zustand
* **Virtualization:** React Virtuoso
